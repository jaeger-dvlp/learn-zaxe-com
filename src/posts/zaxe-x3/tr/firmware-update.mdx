Periyodik olarak, Zaxe X3 ürün yazılımının yeni bir sürümü yayınlanır. Zaxe X3'ünüzün en yeni özelliklere sahip olduğundan emin olmak için,
bellenimi güncel tutmanızı öneririz. Bu, bir ağa veya USB aracılığıyla bağlandığında Zaxe X3 üzerinde gerçekleştirilebilir.
Firmware 5.1.x ve daha eski sürümlerden güncelleme yaparken, önce bir basamaklı firmware sürümü (5.1.94) kurulmalıdır.
Firmware 5.1.x veya daha eski bir sürümden güncelleme yapıyorsanız, aşağıdaki her iki dosyayı da (.tar ve .tar.sig) USB belleğinizin kök klasörüne indirin.

<ColumnCode>
## Yazıcıyı ayarla

Ürün yazılımını güncellemek için cura config için yazıcı modelini ayarlayın. Zaxe X3'ünüzün en yeni özelliklere sahip olduğundan emin olmak için,
bellenimi güncel tutmanızı öneririz.

```bash
$ cura-config --set printer_model "Zaxe-X3"
```

</ColumnCode>

<ColumnCode>

### Kitaplıkları içe aktarın

Mart 2020'de Zaxe, Zaxe X3'ün yeni bir versiyonunu tanıttı. İki versiyonda farklı
farklı ürün yazılımı dosyaları gerektiren elektronik kartlar. Bu sayfadaki iki sürüm arasındaki farkları okuyun.

```py
from cura.ZaxeCodeWriter import ZaxeCodeWriter
from UM.Mesh.MeshWriter import MeshWriter #For the binary mode flag.

from UM.i18n import i18nCatalog
catalog = i18nCatalog("cura")
```
</ColumnCode>

## İşlevleri Tanımlayın

Firmware 5.1.x ve daha eski sürümlerden güncelleme yaparken, önce bir basamaklı firmware sürümü (5.1.94) kurulmalıdır.
Firmware 5.1.x veya daha eski bir sürümden güncelleme yapıyorsanız, aşağıdaki her iki dosyayı da (.tar ve .tar.sig) USB belleğinizin kök klasörüne indirin.

```py
def getMetaData():
    return {
        "mesh_writer": {
            "output": [{
                "extension": "zaxe",
                "description": catalog.i18nc("@item:inlistbox", "Zaxe File"),
                "mode": MeshWriter.OutputMode.BinaryMode,
                "mime_type": "application/zaxe",
                "device_specific": True
            }]
        }
    }

def register(app):
    return { "mesh_writer": ZaxeCodeWriter() }
```
